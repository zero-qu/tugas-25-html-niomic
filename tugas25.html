Mengenai Semantic Element Di HTML 5

  • Articel

    • Ketika kita ingin membuat layout untuk menapilkan block articel-articel yang ada di website kita, tag articel bisa kita gunakan dengan element header dan footer. karena tag articel memiliki type block, artinya element ini memiliki lebar sesuai lebar browswe jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan asid d Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
